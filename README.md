Online Event Registration System

Project Description
The Online Event Registration System is a web-based application that allows users to browse events, register for them, make payments, and receive confirmation. Admins can add, update, and manage events and participant data efficiently.

This project is also integrated with Jira and Confluence for project tracking and documentation.

Objectives
- To provide an easy platform for users to register for events
- To allow administrators to manage event details
- To generate registration confirmation
- To organize project tasks using Scrum methodology

 Features
- User Registration & Login
- Event Browsing
- Event Booking
- Payment Workflow
- Email / Notification Confirmation (conceptual)
- Admin Event Management Dashboard

Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub
- Jira (Scrum Management)
- Confluence (Documentation)

 Project Management
This project was created using the *Scrum model* in Jira:
- 5 Epics created
- Stories and tasks divided
- Sprint planning and tracking
- Work status: To Do, In Progress, Done

Epics:
1. User Module & Registration Flow  
2. Event Catalog & Browsing System  
3. Event Booking & Payment Workflow  
4. Admin Event Management Dashboard  
5. Notification & Email Confirmation  

Confluence Documentation
Three main pages were created in Confluence:
1. Project Plan
2. Task & Effort Estimation
3. Requirements & Meeting Notes
